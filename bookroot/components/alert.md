# Example
```js
var alert = require('blear.components.alert');

alert('字符串消息');
alert(new Error('错误消息'));
```

# `alert(message)`
## `message`
- 类型：`String | Object`
- 说明：需要弹出警告的消息，可以是字符串，或者或者包含 `message` 属性的对象（如 Error 实例）。
- 默认：`""`
