## `.get(statusCode, defaultStatus): status`
根据状态码获取状态值。
```js
httpStatus.get('200');
// => "OK"

httpStatus.get('0000', 'what');
// => "what"
```

## `.set(statusCode, status): map`
根据状态码设置状态值。
```js
httpStatus.set(900, 'yyy');
// => {map}

httpStatus.get(900, 'zzz')
// => "yyy"
```
